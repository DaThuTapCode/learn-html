3: CÁC THUỘC TÍNH CSS CƠ BẢN DƯỚI ĐÂY LÀ 1 SỐ THUỘC TÍNH CSS CƠ BẢN NÊN BIẾT 1 MÀU SẮC (Color) - color: thay đổi màu của văn bản; - background-color: Thay đổi màu nền của phần tử; Ví dụ: p { color: red; background-color: yewllow; } 2 KÍCH THƯỚC VÀ KHOẢNG CÁCH - font-size: Thay đổi kích thước văn bản; - margin: Thay đổi khoảng cách xung quang phần tử; - padding: Thay đổi khoảng cách bên trong phần tử; Ví dụ: p{ font-size: 26px; margin: 10px; padding: 15px; } 3 Chữ (Font) - font-family: Thay đổi kiểu chữ; - font-weight: Thay đổi độ đậm chữ; Ví dụ: p{ font-family: Arial, sans-serif; font-weight: bold; } 4 BỐ CỤC (Layout) -text-align: Căn chỉnh văn bản trái giữa, phải; -border: Thay đổi viền của phần tử.
